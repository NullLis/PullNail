🧱 积木与钉子

一款基于 Unreal Engine 5 的休闲策略游戏
点击积木，放置钉子，用智慧填满每一个角落！

📖 游戏简介

《积木与钉子》是一款轻松上手的 3D 拼图游戏。你需要将不同颜色的钉子附加到各种形状的积木表面，同时遵循严格的规则：

  • 每块积木上只能有 1~6 颗钉子
  • 相邻积木的贴合面上，最多只能有一侧存在钉子
  • 同一种颜色的钉子总数必须是 3 的倍数

游戏在随机生成与逻辑约束之间找到了独特的平衡，既能体验随机的乐趣，又需要运用策略来满足所有规则。

🕹️ 核心玩法

1. 生成积木

游戏启动后会自动生成由多种多边形组成的积木群，每个积木都有六个方向的表面。

2. 附加钉子

钉子会随机选择空闲面进行附加。每种颜色的钉子数量会由全局算法动态平衡，确保满足“3的倍数”约束。

3. 贴合面处理

如果两个积木紧紧相邻，它们的贴合面上不能同时存在钉子。一旦检测到冲突，系统会自动将其中一颗钉子移动到其他空闲面。

4. 全局颜色平衡

所有钉子的颜色分布会在生成结束后进行一次全局调整，确保每种颜色的总数是3的倍数。

🛠️ 技术亮点

技术点描述
引擎Unreal Engine 5.3+
逻辑实现蓝图 (Blueprints) 与少量 C++
碰撞检测基于面法线 + 距离的贴合面识别算法
颜色平衡动态规划算法保证颜色数满足3的倍数
热更新通过 HTTP + JSON 从 Cloudflare Pages 拉取配置文件

⚙️ 配置与更新

游戏支持远程配置更新。配置文件 config.json 托管在 Cloudflare Pages，游戏启动时会自动拉取最新配置。

{
  "version": 5,
  "nail_max": 6,
  "color_pool": ["Red", "Green", "Blue", "Yellow"],
  "block_size": 100
}

修改配置：直接在 GitHub 仓库中更新 config.json 并推送，游戏下次启动即可自动应用新配置。
