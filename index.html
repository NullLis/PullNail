📖 游戏简介

《PullNail》是一款轻松上手的 3D 消除游戏。你只需将每一枚钉子消除，即可过关。

  • 每块积木上只能有 1～6 颗钉子
  • 相邻积木的贴合面上，最多只能有一侧存在钉子。
  • 积木上所有的钉子被消除后，该积木会失去重力掉落。

🕹️ 核心玩法

🖱️

点击钉子

旋转视角找到颜色相匹配并且没有被阻挡的钉子，然后点击它。

🎯

凑满任务

通过找到与上方盒子相同颜色的三枚钉子来完成任务。

🔄

循环往复

循环往复，直至消除场景内所有钉子。

🛠️ 技术亮点

技术点描述
引擎Unreal Engine 5.3+
逻辑实现蓝图 (Blueprints) 与少量 C++
贴合检测基于面法线 + 距离的精准贴合面识别算法
颜色平衡动态规划算法保证颜色数量满足 3 的倍数
热更新通过 HTTP + JSON 从 Cloudflare Pages 拉取配置文件
