📖 游戏简介

《PullNail》是一款轻松上手的 3D 消除游戏。你需要将积木表面的每一枚钉子消除，同时遵循严格的规则：

  • 每块积木上只能有 1～6 颗钉子
  • 相邻积木的贴合面上，最多只能有一侧存在钉子
  • 同一种颜色的钉子总数必须是 3 的倍数

游戏在随机生成与逻辑约束之间找到平衡，既可享受随机带来的新鲜感，又需要运用策略来满足所有规则。

🕹️ 核心玩法

1. 生成积木

自动生成由多种多边形组成的积木群，每个积木拥有多个方向的表面。

2. 附加钉子

钉子随机选择空闲面附加，颜色数量由全局算法动态平衡，保证满足“3的倍数”。

3. 贴合面处理

相邻积木的贴合面上不能同时存在钉子，冲突时自动将钉子移动到其他空闲面。

4. 颜色平衡

生成结束后全局调整颜色分布，确保每种颜色数量均为 3 的倍数。

⚙️ 远程配置

游戏启动时自动拉取配置文件：
https://optimalyuki.github.io/PullNail/config.json

{
  "version": 5,
  "nail_max": 6,
  "color_pool": ["Red", "Green", "Blue", "Yellow"],
  "block_size": 100
}

修改配置：直接更新 GitHub 仓库中的 config.json 并推送，游戏下次启动即可生效。
强制刷新：在 URL 后添加时间戳（如 ?t=20260606）绕过 CDN 缓存。
